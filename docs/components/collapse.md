# Collapse 折叠面板
通过折叠面板收纳内容区域

## 基础用法
可同时展开多个面板，面板之间不影响

<preview path="../demo/Collapse/Basic.vue" title="基础用法" description="Collapse 基础用法"></preview>


## 手风琴效果
每次只能展开一个面板
通过 `accordion` 属性来设置是否以手风琴模式显示。
